# Description
## Text Analyzer
A simple text analyzer system to analyze texts and carry out functions such as: count specific words, identify msot common word, calculate average word length, count paragraphs, and count sentences. It handles errors gracefully if it fails to find the file resource and makes sure not to leave the file open to avoid using too much memory and other bad stuff. 

As I was making this, I realized I was actually building a tokenizer. Albeit a very crude one. It processes the words sentences by sentence, paragraph by paragraph and word by word and can even consider white space and newline characters or completely ignore them. Therefore, this simple silly app actually ended up becoming my gateway tokenizers in a another repository on my github. Feel free to explore. If it's not there, then you're too early:)!!
